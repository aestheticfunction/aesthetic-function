# Security Policy

## Reporting Vulnerabilities

We encourage responsible disclosure of security vulnerabilities in Aesthetic Function.

Do not open public GitHub issues for security vulnerabilities.

Instead, please report them through one of these channels:

- GitHub Security Advisories for this repository
- rdombrowski@gmail.com

Please include a clear description of the issue, affected components or versions, steps to reproduce if available, and any relevant impact details.

We will review reports as promptly as possible and coordinate remediation and disclosure in a professional, responsible manner.
